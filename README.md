# **Project Title**
CodeBase_ReactNative

## **Overview**
This repo contains the **Code Base** of **React Native** with screens and components. We can run it on Android AVD (Android Virtual Device) or by using USB Debugging in a physical phone.

## **Tech Stack**
- React Native
- JDK
- Chocolatey
- Android Studio AVD
- Gradle
- Kotlin
